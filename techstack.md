# Tech Stack – Portfolio Website

## 1. Frontend Stack

### Core Framework

* **React.js (TypeScript)**

  * Strong typing and maintainability
  * Industry standard for scalable UI
  * Aligns with MERN expertise

### Build Tool

* **Vite**

  * Fast development server
  * Optimized production builds
  * Lightweight compared to CRA

### Styling

* **Tailwind CSS**

  * Utility-first styling
  * Rapid UI development
  * Perfect for minimal & glassmorphism design

### UI Patterns

* **Glassmorphism**

  * Backdrop blur
  * Frosted glass cards
  * Subtle shadows and borders

### Routing

* **React Router**

  * Clean route-based navigation
  * SEO-friendly structure (SPA level)

---

## 2. UI / UX Enhancements

### Animations

* **Framer Motion**

  * Smooth transitions
  * Micro-interactions
  * Page and component animations

### Icons

* **Lucide Icons** or **React Icons**

  * Clean, modern iconography
  * Lightweight SVG icons

### Typography

* **Google Fonts**

  * Example: Inter / Poppins
  * Clean corporate readability

---

## 3. State & Utilities

### State Management

* **React Context API**

  * Simple global state
  * Theme (dark/light mode)

### Theme Management

* **Tailwind Dark Mode**

  * Class-based toggling
  * Persistent user preference
